# Stock Management System ```README``` :spiral_notepad:

<br>

## ```About```
- The Stock Management System is a user-friendly web application built using HTML and Java, its purpose is to help users manage their stock more efficiently by removing the need for manual data entry during stock checks and keeping them updated on stock level in real-time. 

<br>

## ```Features```

- ```User Management```
  - [x] List User Accounts
  - [x] Add / Remove User Account
  - [x] Edit User Account
  - [ ] User Registration

- ```Product Management```
  - [x] Add / Remove Product
  - [x] Edit Product
  - [x] Product Quick Search
  - [x] Product ID Barcode Generation / Printing
  - [x] Product Low Stock Auto-notification
  - [x] One-click Product Restock

## ```How to Clone the Repository from GitHub via Eclipse IDE```

> [!IMPORTANT]
> ```Requirements```
>- [GitHub Account](https://github.com/signup)
>- [Eclipse IDE](https://www.eclipse.org/downloads/)
>- Internet Connection <br>

1. Navigate to this [GitHub Repository Address](https://github.com/J-e-r-m-y-n/StockManagementSystemTest) <br>

2. Click on the ```<> Code``` button then ```HTTPS``` and ```copy``` the repository URL <br>
   ![image](https://github.com/user-attachments/assets/d8739cb8-859c-48c6-994f-c80b05bbc4df) <br>

3. Open ```Eclipse IDE``` to import this project: <br>
   - Select ```File > Import``` to open up the Import dialog window
   - Search for ```Git```  then select ```Projects from Git``` and click on the ```Next``` button <br>
   ![image](https://github.com/user-attachments/assets/ebae795d-b076-4946-a723-36715958bdc1) <br>

   - Select ```Clone URI``` and click on the ```Next``` button <br>
     ![image](https://github.com/user-attachments/assets/cad62406-6efd-4ecf-ac4f-90de98b6b9a7) <br>
     
   - Paste the GitHub Repository URL into ```URI (under Location)```, then fill in the ```remaining required information based on this GitHub repository``` and ```your own GitHub authentication credentials``` and click ```Next```
  > [!NOTE]
   > If you have encountered any error after clicking on ```Next```, please ensure that the ```URI``` and ```your own GitHub authentication credentials``` entered are correct. Further troubleshooting steps can be found [here](https://docs.github.com/en/repositories/creating-and-managing-repositories/troubleshooting-cloning-errors).

  - In the Branch Selection window, click on ```Next```, then browse for a ```local directory``` to import the project to and click on ```Next``` again <br>
    ![image](https://github.com/user-attachments/assets/c0e71617-7c17-4945-a981-18ab0481606c) <br>
    ![image](https://github.com/user-attachments/assets/edd4b68a-2777-475e-8e1e-c16098d8ed1d) <br>
  
  - Select ```Import existing Eclipse projects``` and click on ```Next``` then ```Finish``` <br>
    ![image](https://github.com/user-attachments/assets/a522eea5-27e8-4bbf-9643-0d13e668dfb4) <br>
    ![image](https://github.com/user-attachments/assets/e9f0fccf-75a0-4970-a55c-8500e958f2d4) <br>

 - Congratulations! You have successfully cloned this project to Eclipse. :smile:
